## Overview

**GamingWithMe** is a modern, high-performance gaming platform designed to bridge the gap between gamers and service providers. It offers a seamless environment for discovering games, booking sessions, and managing gaming services with a focus on speed and user experience.

## Key Features

- **Dynamic Game Database**: Powered by IGDB integration for real-time game discovery.
- **Service Booking**: Simplified scheduling and management for gaming services.
- **Premium UI/UX**: Built with a "dark mode first" aesthetic using Tailwind CSS and Framer Motion.
- **Secure Architecture**: Robust authentication and database management via NextAuth and Prisma.
